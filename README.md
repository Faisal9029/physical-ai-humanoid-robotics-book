# Physical AI & Humanoid Robotics Book

This comprehensive textbook covers Physical AI and Humanoid Robotics concepts with English and Urdu language support. The book follows the Concept → Architecture → Simulation → Deployment sequence, ensuring every abstraction maps to real hardware or simulation.

## Features

- Complete textbook with 14 chapters across 5 parts
- English (default) and Urdu (secondary) language support with RTL text
- Dark and light mode toggle
- Responsive design optimized for learning
- Internationalization (i18n) ready
- Complete hardware lab architecture guide
- Integrated capstone project for autonomous humanoid implementation

## Book Structure

- **Part I**: The Robotic Nervous System (ROS 2) - Chapters 1-4
- **Part II**: The Digital Twin - Chapters 5-7
- **Part III**: The AI-Robot Brain - Chapters 8-10
- **Part IV**: Vision-Language-Action - Chapters 11-13
- **Part V**: Capstone - Autonomous Humanoid - Chapter 14
- **Appendix**: Hardware & Lab Architecture and Glossary

## Deployment to GitHub Pages

This site is configured for deployment to GitHub Pages. To deploy:

1. Ensure you have the `gh-pages` branch in your repository
2. Run the deploy command:

```bash
npm run deploy
```

This will build the site and push it to the `gh-pages` branch.

Alternatively, you can build the site and manually deploy:

```bash
npm run build
```

The built site will be in the `build` directory and can be served from the `gh-pages` branch.

## Local Development

To run the site locally:

```bash
npm start
```

## Language Support

The site supports both English and Urdu languages:
- English is the default language
- Urdu has right-to-left (RTL) text support
- Language switcher is available in the top navigation bar
