# Physical AI & Humanoid Robotics Textbook

This directory contains the complete textbook content for "Physical AI & Humanoid Robotics".

## Structure

The book is organized into 5 core parts:

1. **Part I: The Robotic Nervous System (ROS 2)** - Covers ROS 2 fundamentals including nodes, topics, services, rclpy agents, and URDF for humanoids
2. **Part II: The Digital Twin** - Focuses on simulation environments with Gazebo physics, Unity visualization, and sensor simulation
3. **Part III: The AI-Robot Brain** - Explores Isaac Sim, Isaac ROS integration, and navigation with Nav2 & SLAM
4. **Part IV: Vision-Language-Action** - Covers voice input with Whisper, LLM planning, and ROS action execution
5. **Part V: Capstone** - Integrates all concepts in an autonomous humanoid project

## Weekly Schedule

The content is designed to be taught over 13 weeks with additional capstone material:

- Weeks 1-3: Part I - Robotic Nervous System
- Weeks 4-6: Part II - Digital Twin
- Weeks 7-9: Part III - AI-Robot Brain
- Weeks 10-12: Part IV - Vision-Language-Action
- Week 13: Part V - Capstone Introduction
- Capstone Project: Extended project integrating all concepts
